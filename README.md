# bzfilter-cli

This is a CLI for identifying which item in the market is a efficient, profitable item for trading in a popular mini-game called _skyblock_ in _hypixel_ which is a popular server in _minecraft_.

# Main features
1. polling features for realtime item details from the API _hypixel skyblock_ provides
2. filter values that can be changed
3. diverse filters such as minimum profit % per, minimum profit $, minimum supply/demand, speed of supply/demand fills, maximum cost and etc
4. editable time between API info polling for accurate average of values
5. basic caching of item infos in a file

# Notes
## Do you even gain profit?
> Its hard to tell how much i gained really means for the people who never played the game but, in about 2 weeks i gained about 35million coins only relying on the items the program suggests to trade. That amount of money is able to buy items that puts you way ahead of players that started playing the game 2 weeks ago.

## Is this program relevant?
> The CLI may become outdated as the meta changes on and on because of the nature of the game having frequent updates.

## My profit isnt what the CLI tells me is! 
> One thing i have also note is that there are taxations for all trades in the game that varies from the upgrades the player has but is NOT accounted for in the filter. Also as i mentioned before the game changes a lot which may have also played a role.
