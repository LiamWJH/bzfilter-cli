This is a CLI for identifying which item in the market is a efficient, profitable item for trading in a popular mini-game called _skyblock_ in _hypixel_ which is a popular server in _minecraft_.

For anyone wondering about the main features, the main features include:
1. polling features for realtime item details from the API _hypixel skyblock_ provides
2. filter values that can be changed
3. diverse filters such as minimum profit % per, minimum profit $, minimum supply/demand, speed of supply/demand fills, maximum cost and etc
4. editable time between API info polling for accurate average of values
5. basic caching of item infos in a file

For those who actually play the game, from early 2026/6 to the half of the month using the CLI i have gained 35 million coins in game.
However the CLI may become outdated or useless as the meta changes on and on and with the nature of the game having frequent updates.
Also the CLI does not have any features for the so called _craft flips_.
One thing i have also note is that there are taxations for all trades in the game that varies from the upgrades the player has but is NOT accounted for in the filter.
Another thing is that although this can make money as i have boasted, it takes consistent effort of refreshing polls, trading same items over and over
that tends to make the person feel bored and tired.
